气泡项目将在这个最简单的博客系统去完善
======

这个系统使用了node的express

代码提交到这https://github.com/zhanghuangang/a-simple-blog

首先大家都先安装node和git

然后使用$git clone https://github.com/zhanghuangang/a-simple-blog.git克隆到自己本地

运行：node app

提交过程：

$git add 你写的文件代码名称

$git commit -m '对它的描述'

$git push origin master 发送你提交在GitHub的分支
